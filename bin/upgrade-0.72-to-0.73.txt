PERL:

install cpan module Date::Manip

AKKADA files:

exchange all lib directory to the new one
exchange all bin directory to the new one
exchange htdocs/css/* files
exchange htdocs/*.js files

add files:
htdocs/img/r_rest.gif
htdocs/img/collapsed.gif
htdocs/img/expanded.gif


AKKADA configuration:

file etc/akkada.conf add:
  'FlapsDeltaMultiplier' => 16,

file etc/conf.d/Web/Rights.conf add:
    'dashboard' => _R_VIE,
    'network' => _R_VIE,
    'form_dashboard_manage' => _R_VIE,

file etc/conf.d/Web/Sections.conf add:
    10 => 'dashboard',

file etc/conf.d/Web/SectionDefinitions.conf add:
      'dashboard' => [
        10,
        ['settings'],
        'dashboard',
      ],


MySQL:

ALTER TABLE `akkada`.`users` ADD COLUMN `dashboard` VARCHAR(255) NOT NULL AFTER `context`;


====

ALL STEPS ABOVE MUST BE COMPLETED BEFORE YOU GO FORWARD

execute (it may take a while): 

cd /akkada/bin
./post_install.sh

to fix statuses errors generated by akkada's software bug execute:

cd /akkada/bin
./entity_status_fix.pl -y
